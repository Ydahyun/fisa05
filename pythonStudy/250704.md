1. 어제 반복문 하고 남은 for문 위주로 배우고 있다. 딕셔너리
```python
# 딕셔너리 items()로 반복문 쓰기

# case 1
fruits = {'딸기': 'Strawberry', '바나나': 'Banana', '사과': 'Apple', '포도': 'Grape'}

for key, val in fruits.items():
    print(f"{key} - {val}")
    
# case 2    
sale1 = { '연필' : [500, 300], '공책': [1200, 2100], '신발':[58000, 2121000]}

for key, [val1, val2] in sale1.items():
    print(key, val1, val2)
```
